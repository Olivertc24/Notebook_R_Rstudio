Instrucciones

Este cuaderno contiene ejercicios prácticos para reforzar los conceptos del Módulo 1.

Tu misión: Escribir el código correspondiente dentro de los bloques (chunks) vacíos para resolver cada problema.

Parte 1: R como Calculadora

Ejercicio 1.1

Calcula el resultado de las siguientes operaciones matemáticas: 1. La suma de 125 más 480. 2. La división de 500 entre 25. 3. 5 elevado a la potencia 4.

Ejercicio 1.2

Usa funciones matemáticas para calcular: 1. La raíz cuadrada de 144. 2. El logaritmo natural de 100. 3. El valor absoluto de -250.

Parte 2: Variables y Asignación

Ejercicio 2.1

  1. Crea una variable llamada precio con el valor 50.
  2. Crea una variable llamada cantidad con el valor 5.
  3. Calcula el total multiplicando ambas variables y guárdalo en una variable llamada total_venta.
  4. Imprime el resultado de total_venta.

Parte 4: Vectores

Ejercicio 4.1

Crea un vector llamado notas que contenga las siguientes calificaciones: 15, 18, 20, 12, 16. Imprime el vector.

Ejercicio 4.2 (Desafío de Coerción)

¿Qué pasa si intentas crear el siguiente vector? mi_vector <- c(10, “Veinte”, 30)

  1. Crea el vector.
  2. Verifica su clase con class().
  3. Explica con un comentario (#) qué ocurrió.

Parte 5: Ejercicio Integrador

Imagina que estás calculando el costo básico de una póliza.

  1. Crea una variable prima_base con valor 1000.
  2. Crea una variable tasa_impuesto con valor 0.16 (16%).
  3. Calcula el monto_impuesto multiplicando la base por la tasa.
  4. Calcula el precio_final sumando la base y el impuesto.
  5. Imprime el precio_final.
