La Filosof√≠a ggplot2

  • ggplot2 es un sistema para crear gr√°ficos de forma declarativa, basado en la ‚ÄúGram√°tica de Gr√°ficos‚Äù. Esto significa que construyes tus visualizaciones especificando componentes en lugar de los pasos exactos para dibujarlos.

  • A diferencia de Excel, aqu√≠ no elegimos un ‚Äúgr√°fico de la galer√≠a‚Äù. Construimos el gr√°fico por capas, como si estuvi√©ramos pintando un cuadro.

La estructura b√°sica es: \[Gr√°fico = Datos + Est√©tica (aes) + Geometr√≠a (geom)\]

  • Gram√°tica de gr√°ficos:

    • Datos (data): El data frame con la informaci√≥n que quieres visualizar.

    • Mapeos est√©ticos (aes()): Define c√≥mo las variables de tus datos se traducen en propiedades visuales (ejes x, y, color, tama√±o, forma, etc.).

    • Geometr√≠as (geom_*): El tipo de gr√°fico que quieres dibujar (puntos, l√≠neas, barras, etc.).

Estructura B√°sica de ggplot2

La estructura fundamental de un gr√°fico en ggplot2 es simple:

    1. Empiezas con la funci√≥n ggplot()
    1. Especificas los datos y los mapeos est√©ticos
    1. A√±ade capas usando +.
ggplot(data = df, aes(x = var1, y = var2)) +
   geom_point()

Preparando los Datos

Vamos a simular una cartera de seguros de Autom√≥vil para los ejemplos.

library(tidyverse)
library(ggplot2)
library(scales)

# Semilla para reproducibilidad
set.seed(123)

cartera <- tibble(
  edad = sample(18:70, 120, replace = TRUE),
  tipo_auto = sample(c("Sedan", "SUV", "Deportivo"), 120, replace = T),
  prima = jitter(edad * 10 + sample(200:500, 120)),
  siniestro = sample(c(0, 1), 120, replace = TRUE, prob = c(0.8, 0.2)),
  mes = rep(seq(as.Date("2025-01-01"), by = "month", length.out = 12), times = 10)
) %>% 
  mutate(siniestro = as.factor(siniestro))

Lineas

Ideal para representar series temporales o la evoluci√≥n de una variable a lo largo de otra continua ordenada.

# 1. Agrupamos y sumarizamos para obtener el total por mes
cartera_resumen <- cartera %>%
  group_by(mes) %>%
  summarise(total_prima = sum(prima))

# 2. Generamos el gr√°fico
ggplot(cartera_resumen, aes(x = mes, y = total_prima)) +
  # Usamos geom_line y geom_point para ver la tendencia
  geom_line(color = "steelblue", linewidth = 1) + 
  geom_point(color = "darkred", size = 2) +
  # Ajustes de formato para el eje de fechas
  scale_x_date(date_labels = "%b", date_breaks = "1 month") +
  labs(
    title = "Evoluci√≥n de Primas Cobradas",
    subtitle = "Total acumulado por mes - A√±o 2025",
    x = "Mes",
    y = "Monto Total de Primas"
  )

* Ajustes manuales con theme(): Permite un control granular sobre cada elemento del tema (texto, l√≠neas, fondos, leyendas).

Las escalas controlan c√≥mo los datos se mapean a los est√©ticos.

Ejes: scale_x_continuous(), scale_y_log10(), scale_x_date(), etc., para controlar rangos, breaks, transformaciones.

Colores/Rellenos: scale_color_manual(), scale_fill_brewer(), scale_fill_viridis_d(), etc., para definir paletas de colores o asignaciones manuales.

   facet_grid(var_fila ~ var_columna): Dos dimensiones, crea una cuadr√≠cula de paneles.

# --- 1. Preparaci√≥n de los datos (Igual que antes) ---
cartera_grafico <- cartera %>%
  group_by(mes, tipo_auto, siniestro) %>%
  summarise(monto_primas = sum(prima), .groups = "drop") %>%
  mutate(siniestro_label = ifelse(siniestro == 1, "Con Siniestro", "Sin Siniestro"))

# --- 2. Gr√°fico 100% ggplot2 Nativo ---
ggplot(cartera_grafico, aes(x = mes, y = monto_primas, fill = siniestro_label)) +
  
  # --- Geometr√≠as ---
  # width = 20 ajusta el ancho de las barras (d√≠as) para que no se toquen entre meses
  geom_col(width = 20, color = "white", alpha = 0.9) + 
  
  # --- Facetas ---
  facet_wrap(~tipo_auto, scales = "free_y") +
  
  # --- Escalas ---
  scale_y_continuous(
    labels = scales::dollar_format(prefix = "$", big.mark = ","),
    # expansion a√±ade un 15% extra arriba para que las etiquetas o barras no toquen el techo
    expand = expansion(mult = c(0, 0.15)) 
  ) +
  scale_x_date(
    date_labels = "%b", 
    date_breaks = "1 month"
  ) +
  scale_fill_manual(
    values = c("Sin Siniestro" = "#2C3E50", "Con Siniestro" = "#E74C3C")
  ) +
  
  # --- Etiquetas ---
  labs(
    title = "An√°lisis de Primas Emitidas por Segmento",
    subtitle = "Comparativa mensual de p√≥lizas seg√∫n estado de siniestralidad",
    x = NULL,
    y = "Primas Cobradas (USD)",
    fill = "Estado de P√≥liza",
    caption = paste0("Fuente: Datos simulados | Ejecuci√≥n: ", Sys.Date())
  ) +
  
  # --- Tema Personalizado ---
  theme_minimal(base_size = 12) +
  theme(
    # T√≠tulos y Textos (Usando element_text nativo)
    plot.title = element_text(face = "bold", size = 16, color = "#2c3e50", hjust = 0),
    plot.subtitle = element_text(size = 12, color = "grey40", margin = margin(b = 10)),
    plot.caption = element_text(color = "grey60", face = "italic", margin = margin(t = 15)),
    
    # Leyenda movida abajo para dar m√°s espacio al gr√°fico
    legend.position = "bottom",
    legend.title = element_text(face = "bold", size = 10),
    
    # Estilo de las facetas (Cajitas grises arriba de cada gr√°fico)
    strip.background = element_rect(fill = "#ECF0F1", color = NA),
    strip.text = element_text(face = "bold", color = "#2C3E50", size = 11),
    
    # Limpieza de Ejes
    axis.text.x = element_text(angle = 45, hjust = 1, size = 10), # Rotaci√≥n para leer los meses
    axis.line.x = element_line(color = "grey80"), # L√≠nea sutil en el eje X
    panel.grid.minor = element_blank(), # Eliminar cuadr√≠cula menor
    panel.grid.major.x = element_blank() # Eliminar l√≠neas verticales para limpieza visual
  )
